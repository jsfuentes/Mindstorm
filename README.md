# Mindstorm

This will likely change in the next few months. But for large projects right now, it is faster for you to create the high level spec and decisions then for
AI to create it and you to iterate on it. AI should still help you by providing context and asking clarifying questions. The output of such a tool should be 
ready to be feed to Claude for production.

Features:
- Minimal UI so you can think, but AI can help
- Create a new spec specifying a local github repo
- Speak and send messages which will be organized into a coherent document
- Questions on right auto update with claude code generated questions based on how it would implement it in the provided repo
- Spin up multiple claude code agents for plan mode/context gathering tasks on the left with colors indicating their status
